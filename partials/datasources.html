Datasources

Refugee and Asylum Seekers

UNHCR Dataset

Global dataset on refugee and asylum seekers was obtained from the UNHCR Population Statistics website

Global Migration Patterns

OECD Dataset

Global data on migration patterns was downloaded from the International Migration Database maintained by the OECD on their website

Global Development Indicators

World Bank Dataset

Country-level dataset of global development indicators for the last decade (from 2003 to 2013) downloaded from the World Bank data library

Zimbabwe Story Content

BBC Zimbabwe Profile

Content for the Zimbabwe story is derived from the Zimbabwe profile and, specifically, timeline listed on the BBC website.

Data Cleaning

After obtaining the above data sets, we consolidated the data into two PostgreSQL tables (migrant/asylum seeker flow and indicators) using a combination of shell scripts, python code and sql statements.
